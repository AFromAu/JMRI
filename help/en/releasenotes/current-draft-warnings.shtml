• This test release, and all subsequent ones, require the use of Java 11 or later. Java 8 is explicitly not supported. There's more information on the JMRI Tech Road Map and JVM Capabilities pages.
  • A number of classes and methods that have been deprecated for a long time are removed in this release. These include:
    • apps.JmriPlugin
