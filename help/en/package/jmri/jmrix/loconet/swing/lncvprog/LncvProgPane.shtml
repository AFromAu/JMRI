LocoNet LNCV Programming Pane

The "Configure LNCV Modules" tool lets you access and program all connected LocoNet devices that respond to the (Uhlenbrock) LNCV protocol.
You find the tool in the LocoNet menu as long as a LocoNet connection is available.

As the tool is still under development, we advice to use it in combination with the "Show Raw Data" option in the LocoNet Monitor to check for any additional responses rejected by the LNCV filters.

Simply click "Open Programming Session" to start a session.

To close a session, click "Leave Programming Session".

Unless a session is running, LNCV modules will not respond to Read/Write commands.

LNCV Programming requires entry of an Article (model) number provided in the manufacturer's documentation.
Enter the number eg. 5033 in the "Article" field before proceeding.

After opening a session, the "Open Module Programming" is active. To close the programming of a module, click the same button, that now reads "Stop Module Programming".

Responses will be listed in the table (under development).
